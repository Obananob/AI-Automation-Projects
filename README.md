## Crypto Watcher

**Description:** An automation built with Make.com that monitors cryptocurrency prices and sends alerts for significant changes.  
**Tools Used:** Make.com, CoinGecko API

## Customer Support Agent

**Description:** A workflow in n8n that automates customer support inquiries and responses.  
**Tools Used:** n8n

## Data Summarizer

**Description:** Uses OpenAI to generate concise summaries of provided data or text.  
**Tools Used:** OpenAI API

## Email Automation for Student Notifications

**Description:** Automates sending targeted emails to students in a selected class based on form submissions, updating Google Sheets, and looping through recipients.  
**Tools Used:** n8n, Google Sheets

## Invoice Flow

**Description:** Detects incoming invoices via email, uploads them to Google Drive, and notifies the billing team with a link.  
**Tools Used:** n8n, Google Drive

## Personal Assistant

**Description:** Provides daily summaries of incoming emails for a quick overview.  
**Tools Used:** n8n

## Product Image Expert

**Description:** Converts raw product images into market-ready visuals using AI and sends the final image to the customer.  
**Tools Used:** n8n, OpenAI Image Edit Model, Google Drive

## School Library Management System

**Description:** An Airtable-based system for managing library books, students, and transactions, including automated notifications for book borrowing.  
**Tools Used:** Airtable

## YouTube Comment Scraper

**Description:** An automation that scrapes comments from a specified YouTube video URL and stores them in Airtable.  
**Tools Used:** n8n, Apify, Airtable

## Google Maps List Scraper

**Description:** Scrapes business listings from Google Maps based on a given location and keyword, filtering out closed businesses and appending results to Google Sheets.  
**Tools Used:** n8n, Apify, Google Sheets
